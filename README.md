A little spreadsheet in OCaml + Bucklescript

built on [bucklescript-tea](https://github.com/OvermindDL1/bucklescript-tea) and a [starter kit](https://github.com/tcoopman/bucklescript-tea-starter-kit)


## Install

```
npm install
```

## Build

```
npm run build
```

## Watch

```
npm run watch
```
